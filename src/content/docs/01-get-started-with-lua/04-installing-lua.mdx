title: Instalación de Lua
description: Lua es un lenguaje de programación dinámico y ligero diseñado para ser integrado dentro de otras aplicaciones.
---

Scoop es un gestor de paquetes para Windows que simplifica la instalación de herramientas de desarrollo y aplicaciones, eliminando la necesidad de configuraciones complejas. En este artículo, aprenderás cómo instalar Lua utilizando Scoop paso a paso.

### Requisitos previos

Antes de comenzar, asegúrate de tener lo siguiente:

1. **Windows 10/11** o una versión compatible.
2. **PowerShell** habilitado.
3. Conexión a Internet.

### Paso 1: Instalar Scoop

Sigue estos pasos para instalar Scoop:

1. Abre PowerShell como administrador.
2. Ejecuta el siguiente comando para habilitar la ejecución de scripts:
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
3. Instala Scoop ejecutando:
   ```powershell
   irm get.scoop.sh | iex
   ```
   Este comando descarga e instala Scoop en tu sistema.

4. Verifica que Scoop está instalado correctamente ejecutando:
   ```powershell
   scoop --version
   ```

### Paso 2: Configurar Scoop (Opcional)

Por defecto, Scoop instala herramientas en el directorio `~/scoop`. Si necesitas personalizar la ubicación o agregar soporte para aplicaciones con dependencias extra, puedes instalar el bucket `extras` con:

```powershell
scoop bucket add extras
```

### Paso 3: Instalar Lua

1. Instala Lua ejecutando:
   ```powershell
   scoop install lua
   ```
2. Una vez completada la instalación, verifica la versión de Lua para confirmar que se instaló correctamente:
   ```powershell
   lua -v
   ```
   Deberías ver una salida similar a:
   ```
   Lua 5.4.4  Copyright (C) 1994-2023 Lua.org, PUC-Rio
   ```

---

### Paso 4: Usar Lua desde la terminal

Puedes empezar a usar Lua directamente desde la terminal de PowerShell escribiendo:

```powershell
lua
```
Esto abrirá el entorno interactivo de Lua (REPL), donde puedes ejecutar comandos como:

```lua
print("¡Hola, Lua!")
```

Para salir del entorno REPL, simplemente escribe `os.exit()` o presiona `Ctrl + Z` seguido de `Enter`.

---

### Paso 5: Administrar Lua con Scoop

Scoop facilita la administración de versiones y actualizaciones de Lua:

- **Actualizar Lua:**
  ```powershell
  scoop update lua
  ```

- **Actualizar todos los paquetes instalados:**
  ```powershell
  scoop update *
  ```

- **Desinstalar Lua:**
  ```powershell
  scoop uninstall lua
  ```
  
### Ventajas de usar Scoop para instalar Lua

1. **Simplicidad:** Instalar y actualizar Lua con un solo comando.
2. **Mantenimiento:** Scoop gestiona las dependencias y mantiene tu sistema organizado.
3. **Portabilidad:** Los paquetes instalados no requieren configuraciones adicionales.

### Conclusión

Instalar Lua con Scoop es un proceso rápido y eficiente que elimina la necesidad de realizar configuraciones manuales. Gracias a Scoop, puedes concentrarte en desarrollar proyectos con Lua sin preocuparte por configuraciones complejas. ¡Empieza a explorar Lua hoy mismo!

Si necesitas más información sobre Scoop, visita su [documentación oficial](https://scoop.sh).
