Las **variables** son elementos fundamentales en cualquier lenguaje de programación, incluyendo Lua. Son espacios de almacenamiento que permiten guardar datos que pueden ser utilizados y manipulados a lo largo del programa. Lua, al ser un lenguaje dinámico y sencillo, facilita la creación y el uso de variables.

## ¿Qué son las variables?

En términos simples, una **variable** es un nombre simbólico que representa un valor en memoria. Este valor puede ser de cualquier tipo de datos: un número, una cadena de texto, una tabla o incluso una función. Las variables se utilizan para almacenar información que puede ser cambiada o utilizada en diferentes partes de un programa.

En Lua, no es necesario especificar un tipo para la variable, ya que Lua es un lenguaje **dynamically typed** (tipado dinámico). Esto significa que una variable puede cambiar su tipo a lo largo del tiempo. 

## Declaración e Inicialización de Variables

**Declaración de Variables**

En Lua, la declaración de variables es muy simple. No necesitas especificar el tipo de la variable, solo el nombre y, opcionalmente, su valor inicial. El nombre de una variable debe comenzar con una letra o un guion bajo (`_`), seguido de letras, números o guiones bajos. Lua no permite nombres de variables que empiecen con un número.

La sintaxis básica para declarar una variable en Lua es la siguiente:

```lua
nombre_de_variable
```

Por ejemplo:

```lua
edad
nombre
```

Estas declaraciones definen las variables `edad` y `nombre`, pero no les asignan ningún valor.

**Inicialización de Variables**

Cuando declaras una variable, puedes inicializarla, es decir, asignarle un valor de forma inmediata. Para ello, utilizas el operador de asignación `=`. La sintaxis para inicializar una variable es la siguiente:

```lua
nombre_de_variable = valor
```

Por ejemplo:

```lua
edad = 25
nombre = "Juan"
```

En este caso, la variable `edad` es inicializada con el valor `25` y la variable `nombre` con el valor `"Juan"`.

**Declaración e Inicialización en una sola línea**

Es común declarar e inicializar variables en una sola línea, lo cual es práctico y eficiente en muchos casos. Ejemplo:

```lua
numero = 10
mensaje = "Hola Mundo"
```

## Cambiar el Valor de las Variables

Una de las ventajas de las variables en Lua es que su valor puede cambiar en cualquier momento del programa. Para modificar el valor de una variable, simplemente asignamos un nuevo valor:

```lua
edad = 30
```

Este cambio modificará el valor de la variable `edad` de `25` a `30`.

**Ejemplo**

A continuación, un ejemplo que muestra cómo declarar, inicializar y utilizar variables en Lua:

```lua
-- Declaración de variables
local nombre = "Ana"
local edad = 22
local altura = 1.65

-- Usar las variables
print("Nombre: " .. nombre)
print("Edad: " .. edad)
print("Altura: " .. altura .. " metros")
```

**Salida:**

```
Nombre: Ana
Edad: 22
Altura: 1.65 metros
```

En este ejemplo, hemos declarado tres variables: `nombre`, `edad` y `altura`. Luego, las imprimimos utilizando la función `print()`, concatenando los valores con el operador `..`.
