## Tipos de Datos en Lua

En Lua, los tipos de datos son fundamentales para definir y manipular la información en un programa. Lua es un lenguaje de programación dinámico, lo que significa que las variables no tienen un tipo fijo y pueden cambiar de tipo a lo largo de la ejecución del programa. A continuación se detallan los principales tipos de datos en Lua.

## 1. Números (Numbers)
Los números en Lua son representados como valores de punto flotante (doble precisión de 64 bits), por lo que pueden ser tanto enteros como decimales.

**Ejemplo:**
```lua
numero_entero = 42
numero_decimal = 3.14
```

## 2. Cadenas de Texto (Strings)
Las cadenas de texto en Lua son secuencias de caracteres encerradas entre comillas dobles (`" "`) o simples (`' '`).

**Ejemplo:**
```lua
mensaje = "Hola, Mundo!"
```

## 3. Booleanos (Booleans)
El tipo **booleano** se utiliza para representar valores de verdad. Los dos valores posibles son **`true`** y **`false`**.

**Ejemplo:**
```lua
es_adulto = true
es_nino = false
```

### 4. **Nil**
El tipo **`nil`** representa un valor vacío o no asignado. Se utiliza para indicar que una variable no tiene un valor.

**Ejemplo:**
```lua
variable = nil
```

Estos son los principales tipos de datos en Lua, los cuales permiten una gran flexibilidad en el manejo de información y la creación de estructuras complejas.
